Java Script

Welcome to the exciting world of java script! Learning to utilize this language will allow you to create an interactive webpage with minimal effort through event-based programming. Here are some of the key abilities of java script:

  • Simpler language to learn as it uses the DOM.
  • Works fast on the users end.
  • Allows extended functionality with third party add-ons.
  • Easy to debug and test.

Swift

Get sweeped off your feet with swift! Designed to be anyone's first programming language swift's readability and functionality made it the replacement for Objective-C with programming Apple products. Here are some benefits to using swift:

  • Offers a safe platform for developing to diminish crashes and problematic behavior.
  • Supports dynamic libraries so you can link newer versions of Swift as it evolves.
  • Offers 'Playgrounds' so the creater can test out their program without creating an entire app.
  • Syntax resembles plain English making it easier for new programmers.
